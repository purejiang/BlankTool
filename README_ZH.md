# BlankTool

## 简介：
使用 pyside2 开发的桌面程序，集合了各种 android 开发的小功能，通过桌面化的 UI 简化了各种工具的使用。


## 说明：

### 功能
该桌面程序已包含以下功能：
- apk / aab 的安装
- apk 的解析
- apk 反编译

正在开发的功能：
- 目录重编为 apk
- apk 重签
- 手机上已安装的 apk 列表展示+提取

未来规划的功能：
- aab 解析
- aab 的 assets 资源分割
- 方法数统计\
...

### 工具+环境
- apktool
- bundletool
- aapt2
- adb
- java 1.8
